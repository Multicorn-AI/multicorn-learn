# Drafts

This directory holds **outline drafts** generated by the multicorn-content agent.
These are working documents - not published content.

## What's in here

Every Sunday at 6pm UTC, the content agent fetches AI news from RSS feeds (The
Neuron, MIT Tech Review AI, The Verge AI), filters articles for relevance to
Multicorn Learn topics, and generates up to 3 article outlines per week. Each
outline opens as a PR titled `Draft outline: <title>` with a markdown file in
this directory.

## What an outline contains

- Frontmatter: title, description, date, audience level, source URL
- Five section headings

That's it. No body content. The outline is a starting point for a human writer.

## Workflow

1. Review the PR - skim the title, summary, and section headings
2. **Merge** if it's something worth writing about - the draft lands in `drafts/`
   for you to pick up later
3. **Close** if it's not a good fit - nothing happens, no content created

## Writing the article

When you're ready to turn an outline into a full article:

1. Create a new branch from main
2. Move the file from `drafts/` to `content/blog/` (or wherever published
   articles live)
3. Expand the outline into a full article
4. Update the sitemap and blog index
5. Open a PR for the finished article

## What this directory is NOT

- Not published content. Files here never appear on the Learn site.
- Not a queue of articles to write. Outlines that sit here for months are fine
  to delete if they're stale.
- Not edited by the agent after creation. The agent only creates new outlines;
  it never modifies existing files.

## How to disable the agent

If you need to pause the content agent, disable the `Content Agent` workflow in
GitHub Actions, or remove the `schedule` trigger from
`.github/workflows/content-agent.yml`.
